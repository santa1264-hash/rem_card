# UI-стандарты оперблока

## Скроллбар

Эталонный скроллбар оперблока - скроллбар правой панели быстрых назначений. Все новые скроллбары оперблока должны использовать общий стиль из `ui/operblock_view/operblock_control_styles.py`.

- фон области скроллбара прозрачный;
- трек: `#e8edf3`, граница `#d4dce7`, радиус `4px`;
- ручка: `#8fa3ba`, граница `#71869d`, радиус `4px`;
- hover ручки: `#7890aa`;
- штатные стрелки `add-line` / `sub-line` скрыты;
- вертикальная ручка имеет минимальную высоту `28px`;
- горизонтальная ручка использует те же цвета и визуальный вес.

## Стрелочные переключатели времени

Эталон - кнопки вверх/вниз в блоке "Время начала" диалогов изменения препарата и газа.

- фон: `#EAF3FF`;
- граница: `#BFD1E5`;
- hover: фон `#DCEBFA`, граница `#9FB8D2`;
- pressed: `#CFE3F7`;
- радиус: `5px`;
- иконка: `icon/combo_arrow_down.svg`, для кнопки вверх SVG поворачивается на 180 градусов;
- размер может немного меняться под контекст, но визуальная форма, цвета, радиус и стрелка должны сохраняться.

## Кнопка раскрытия выпадающего списка

Кнопка справа у `QComboBox` в оперблоке использует тот же стрелочный стандарт, что и переключатели времени: светло-синий фон, граница, радиус `5px` и `combo_arrow_down.svg`. Стиль берётся из `operblock_combo_box_style()`.

Для новых элементов не копировать QSS вручную. Использовать:

- `operblock_vertical_scrollbar_style()`;
- `operblock_horizontal_scrollbar_style()`;
- `operblock_arrow_button_style()`;
- `operblock_arrow_icon()`;
- `operblock_combo_box_style()`.
