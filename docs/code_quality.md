# Проверки качества кода

Быстрый локальный gate:

```powershell
python scripts\code_quality_checks.py
```

Он проверяет:

- `flake8 --select=F821`: неопределенные имена запрещены полностью.
- BOM scan: UTF-8 BOM в файлах репозитория запрещен полностью.
- `radon cc --min F`: новые блоки с оценкой сложности `F` запрещены.

Текущие `F`-блоки занесены в baseline внутри `scripts/code_quality_checks.py`.
Это временный список: существующие сложные методы рефакторятся отдельными задачами, а новые `F` появляться не должны.

На текущем этапе не требуется снижать всю сложность до `D` или `C`, чтобы не создавать шум в большом существующем коде.
